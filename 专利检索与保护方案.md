## 1. 检索范围与限制

### 1.1 已尝试访问的官方入口

用户指定的国知局专利检索入口：

- https://pss-system.cponline.cnipa.gov.cn/conventionalSearch

本次直接访问该入口时出现超时，无法在当前环境中完成交互式检索。

为确认官方来源，本次参考了：

- 国家知识产权公共服务平台：https://ggfw.cnipa.gov.cn/
- 国知局专利检索及分析系统用户手册：https://pss-system.cponline.cnipa.gov.cn/help/专利检索及分析系统用户使用手册-检索.pdf
- WIPO 对 CNIPA Patent Search and Analysis System 的说明：https://inspire.wipo.int/patent-search-and-analysis-system

国知局用户手册显示，常规检索支持“自动识别、检索要素、申请号、公开（公告）号、申请（专利权）人、发明人、发明名称”等字段；高级检索支持申请号、申请日、公开日、发明名称、摘要、权利要求、说明书、关键词等字段。因此，后续正式检索建议由专利代理人在国知局系统中复核本文列出的公开号、申请人、权利要求和法律状态。

## 2. 我们拟保护的核心技术方案

当前产品不应把“AI 批改作业”作为主保护点。该功能过于宽泛，已有大量公开产品和专利。

更建议保护以下组合方案：

```text
便携折叠摄像头硬件
→ 黑色底板提供稳定高对比背景
→ 自动识别纸张边界并裁剪矫正
→ 老师拍摄答案卷并做简单标记
→ 系统识别标记生成题目区域模板
→ 学生卷复用答案卷模板定位每题答题区域
→ OCR/公式符号识别转文本
→ 选择题/判断题/简单填空规则判分
→ 复杂题调用通用大模型
→ 前 3-5 张老师修正作为同批次校准样本
→ 生成电子批改图层和成绩数据
```

## 3. 建议申请的一组专利

### 3.1 发明专利 1：基于黑色底板的便携作业图像自适应裁剪方法

保护目标：

- 保护“黑色底板 + 任意规格纸张 + 自动边界识别 + 透视矫正”的硬件算法协同方案。

建议权利要求核心：

- 摄像头拍摄包含黑色底板和纸张的原始图像。
- 根据纸张与黑色底板的颜色、亮度或边缘对比，识别纸张轮廓。
- 基于轮廓顶点生成纸张区域。
- 对纸张区域进行旋转、透视和裁剪。
- 输出标准化答卷图像。

注意：

- 不要只写“识别试卷边缘并裁剪”，已有公开专利覆盖拍照试卷边缘检测和透视变换。
- 要强调“便携阅卷仪的统一深色底板作为视觉参照物”，以及“不依赖特定答题卡或固定纸张规格”。

### 3.2 发明专利 2：基于答案卷标记的题目区域模板生成与学生卷映射方法

保护目标：

- 这是目前最值得优先保护的技术点。
- 保护“老师在答案卷上做简单标记，系统建题目区域模板，学生卷复用模板”的流程。

建议权利要求核心：

- 获取答案卷图像。
- 识别老师在答案卷上做出的标记线、框选区域或颜色标记。
- 基于标记和网格划分生成题目区域模板。
- 绑定题号、分值、评分标准和区域坐标。
- 获取同版学生卷图像。
- 将答案卷模板映射到学生卷，切分每题答题区域。

规避价值：

- 区别于普通“自动识别题目区域”的已有技术。
- 区别于固定答题卡、题库模板或预制版式。

### 3.3 发明专利 3：低成本题型分流与混合判分方法

保护目标：

- 保护“规则判分 + 大模型判分”的成本优化调度链路。

建议权利要求核心：

- 对每题识别结果进行题型分流。
- 选择题、判断题、简单填空题进入规则判分。
- 复杂填空、简答、作文、复杂计算题进入大模型评分。
- 根据 OCR 置信度、题型和答案复杂度决定是否调用大模型。
- 汇总不同评分路径的结果形成统一批改结果。

注意：

- 不要宣称“题型分流”本身是全新，已有专利涉及题型识别和客观题自动判分。
- 重点写“为了降低通用大模型 API 成本，先以题目区域 OCR 文本作为输入，按题型和置信度分流”。

### 3.4 发明专利 4：基于老师前置样本修正的同批次评分校准方法

保护目标：

- 保护前 3-5 张校准样本机制。
- 这是区别于普通自动批改和普通大模型批改的重要闭环。

建议权利要求核心：

- 系统先批改前若干份学生答卷。
- 老师修改 AI 分数、评语或错误类型。
- 系统将老师修正样本转化为本次任务的评分校准参数。
- 后续同一批答卷调用大模型时，将校准参数作为输入。
- 同一题型或同一题目的后续评分受校准样本影响。

保护重点：

- “同一批次/同一套试卷/同一老师评分标准”的短周期校准。
- 不必主张长期个性化模型训练，降低与模型训练类专利的重叠。

### 3.5 实用新型专利：便携折叠摄像头阅卷硬件结构

保护目标：

- 保护硬件外形和结构。

建议保护点：

- Head 摄像头组件。
- 可伸缩支架杆。
- 可折叠连接结构。
- 黑色底座板。
- Type-C 或 WiFi 摄像头传输组件。
- 底座上辅助定位线或纸张放置区。

注意：

- 高拍仪、折叠支架、伸缩杆已有大量公开方案，单独保护强度有限。
- 应把“用于作业/试卷自适应裁剪的黑色底座板”和“便携 ToC 阅卷场景”写进结构功能关系。

### 3.6 外观设计专利：整机外观

保护目标：

- 防止白牌厂商直接照抄外观。

建议保护：

- 折叠状态外观。
- 展开状态外观。
- Head、支架、底板整体比例。
- 底板定位线样式。

## 4.4 家庭/移动场景与教师个人使用的可保护性

### 4.4.1 初步判断

“家庭场景”“移动场景”“教师个人使用”“便携式智能改卷”这些应用场景本身可以写入专利背景、技术问题、使用场景和部分权利要求限定中，但不适合作为唯一核心创新点。

原因：

- 专利保护的不是抽象商业场景，而是解决技术问题的具体技术方案。
- “给老师个人使用”“家庭或移动场景使用”容易被认为只是使用对象或应用场景变化，创造性较弱。
- 已有公开专利和产品已经覆盖了家庭学习场景、移动拍照批改、教师端作业批改、便携阅卷设备等宽泛方向。

因此，更建议将该场景作为限定条件，与具体技术方案组合保护：

```text
适用于家庭或移动教学场景的便携式作业批改设备及方法
→ Type-C/WiFi 摄像头
→ 可折叠支架与黑色底板
→ 多形态、多介质纸质作答载体自适应裁剪
→ 答案卷标记建题目区域模板
→ 学生卷模板映射
→ OCR 文本化
→ 题型分流与大模型批改
→ 前 3-5 张教师校准反馈
→ APP 电子批改图输出
```

### 4.4.2 可尝试保护的写法

建议在发明专利或实用新型中加入以下限定：

- 设备适用于家庭、办公室、教室外出、移动授课、培训机构巡回教学等非固定阅卷室场景。
- 设备面向教师个人携带和使用，不依赖学校机房、专用扫描仪或固定阅卷系统。
- 设备通过可折叠结构和可伸缩支架形成便携式拍摄状态。
- 使用黑色底板在非固定环境中提供稳定拍摄背景。
- 通过 APP 完成答案卷录入、学生卷批改、老师修正和电子批改图查看。

注意：

- 这些限定可以帮助区别于 B 端大型阅卷机、智能台灯、打印机批改系统和专用答题卡读卡器。
- 但不能只靠这些限定获得强保护，必须与黑色底板裁剪、答案卷标记模板、同批次校准等技术方案绑定。

### 4.4.4 对我们方案的结论

家庭/移动场景和教师个人 ToC 使用可以作为专利方案的重要限定，但不宜单独主张。

建议申请时采用以下定位：

- “一种适用于家庭或移动教学场景的便携式作业批改设备及方法”作为标题或背景方向。
- 权利要求核心落在“可折叠摄像头 + 黑色底板 + 答案卷标记模板 + 学生卷映射 + 题型分流 + 教师校准反馈”。
- 用该场景区别大型校用阅卷机、智能台灯、读卡器、打印机批改系统和纯 APP 批改系统。

可保护性判断：

- 宽泛场景：弱。
- 场景 + 具体硬件结构：中。
- 场景 + 具体硬件结构 + 黑色底板裁剪 + 答案卷标记模板 + 同批次教师校准：中高。

换言之，我们不能垄断“教师个人便携改卷”，但可以尝试保护“为了让教师个人在家庭或移动场景中处理多形态、多介质纸质作答载体而设计的这一整套低成本硬件与算法流程”。

## 5. 已发现的高相关专利与侵权风险点

以下为公开文本初筛，风险等级是产品角度初判，不代表法律结论。

### 5.1 CN114973253A：作业批改与显示方法、装置、设备及存储介质

来源：

- https://patents.google.com/patent/CN114973253A/zh
- PDF：https://patentimages.storage.googleapis.com/54/6e/e6/0c68731f8ebae4/CN114973253A.pdf

申请人：

- 科大讯飞相关主体。

相关公开点：

- 摄像头组件获取目标区域图像。
- 云端服务器对图像进行信息识别与批改后生成批改痕迹图像。
- 批改符号位置与题目位置匹配。
- 可生成学情报告。
- 公开权利要求中包含投影显示学情报告/批改痕迹的方案。

风险点：

- “摄像头拍作业 + 云端批改 + 返回批改痕迹图像”与我们部分链路高度相似。
- 若我们做投影回纸面，风险显著上升。
- 如果只在 APP 中生成电子批改图，不做投影，直接落入其公开主权利要求的风险下降，但仍需看最终授权权利要求。

建议规避：

- 不做投影模块。
- 不宣传“投影批改痕迹”。
- 结果叫“电子批改图层”或“APP 批改预览”，不要模拟其投影场景。
- 强化黑色底板、答案卷标记模板、题型分流和校准样本。

风险等级：

- 中高。

### 5.2 CN109712456A：一种基于摄像头的学生纸质作业智能批阅系统

来源：

- https://patents.google.com/patent/CN109712456A/zh

相关公开点：

## 6. 侵权风险重点标注

### 6.1 高风险功能点

1. 标准卷/答案卷区域模板直接映射到学生卷并识别字符比对。

原因：

- CN109815932B 已公开“标准试卷图像标注框位置映射到待批改试卷”的方案。

规避：

- 采用老师主动标记、网格划分、题目区域模板和评分标准绑定。
- 不只做标准答案字符比对。

2. 摄像头拍作业后生成批改痕迹图像并投影回纸面。

原因：

- CN114973253A 公开了云端生成批改痕迹图像与投影显示路径。

规避：

- 不做投影。
- 仅在 APP 中生成电子批改图。

3. 拍照试卷含背景，使用通用边缘检测模型裁剪试卷区域。

原因：

- CN113112511A/B 已覆盖原始图像含背景、边缘检测、透视变换、试卷区域图像。

规避：

- 依靠黑色底板的稳定对比背景、定位线、角点参照物、底板校准点形成更具体方案。

4. 题型识别后选择/填空/判断直接比对，主观题按给分标准判分。

原因：

- CN112116840A/B、US11508251B2 等已有相近公开。

规避：

- 把题型分流定位为成本优化子模块。
- 核心放到大模型调用前的 OCR 文本化、调用成本控制、老师同批次校准。

### 6.2 中风险功能点

1. 手写 OCR 转文本后批改。
2. 答题区域规则切图后比对标准答案。
3. 云端服务器批改后返回解析内容。
4. 生成学情报告和错题统计。

规避：

- 这些功能可做，但不作为核心专利主张。
- 在产品上注意不要照搬竞品的“原卷留痕打印”“投影批改痕迹”等具体表达。

## 7. 建议调整后的产品实现边界

### 7.1 建议保留

- Type-C 摄像头版本和 WiFi 摄像头版本。
- 黑色底板作为统一拍摄背景。
- 答案卷红笔/颜色标记辅助题目区域模板。
- 老师输入分值、给分点、扣分规则。
- 选择题、判断题和简单填空规则判分。
- 复杂题走大模型。
- 前 3-5 张老师修正校准同批次评分。
- APP 电子批改图输出。

### 7.2 建议谨慎

- “标准答案框直接映射学生卷框”。
- “批改痕迹图像”这一术语。
- “原卷留痕”这一术语。
- “学情报告即时生成”作为核心宣传点。

### 7.3 建议暂不做

- 投影批改结果到纸面。
- 扫描、批改、打印一体机。
- 依赖专用答题卡。
- 固定题库或预制教辅版式。

## 9. 初步结论

### 9.1 能保护什么

可以尝试保护：

- 低成本 ToC 便携摄像头阅卷仪的硬件算法组合。
- 黑色底板辅助任意规格纸张自裁剪。
- 老师在答案卷上做简单标记生成题目区域模板。
- 答案卷模板映射学生卷，并绑定评分标准。
- OCR 后题型分流，减少大模型 API 调用。
- 前 3-5 张老师修正样本校准同批次评分。

### 9.2 不能强保护什么

不宜作为核心保护：

- AI 自动批改。
- 摄像头拍作业。
- 手写 OCR。
- 题目区域识别。
- 客观题与标准答案比对。
- 主观题按给分标准批改。
- 批改结果显示或生成学情报告。

这些方向已有大量公开技术。

### 9.3 最大侵权风险

当前最大风险点是：

1. 标准卷/答案卷模板映射学生卷区域，需重点规避 CN109815932B。
2. 云端生成批改痕迹图像，需规避 CN114973253A 的投影和批改痕迹路径。
3. 拍照试卷背景裁剪，需规避 CN113112511A/B 的通用边缘检测裁剪路径。
4. 题型分流与简单题判分，需规避 CN112116840A/B、US11508251B2 等宽泛自动批改方案。
